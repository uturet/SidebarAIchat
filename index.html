Side panel on any site

Open the Chrome side panel and chat without leaving the page. Messages stream with code highlighting for readable outputs.

Bring your own providers

Connect assistants like OpenAI (Chat Completions) or Ollama. Configure available models per provider, then choose one for each chat.

Planned: additional providers (e.g., Claude, Grok).

Reusable chat configurations

Bundle a provider, model list, and custom instructions into a configuration you can reuse. Swap setups instantly across chats.

Work with files

Drag and drop files into the panel to include their contents in context. Text formats like TXT, CSV, JSON, and XML are supported first.

Local-first data

Chats, assistants, and configurations are stored locally in your browser. You control the keys and the data.
