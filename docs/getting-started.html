Getting Started

Follow these steps to move from installation to your first successful conversation.

1. Install and open the extension

Install Sidebar AI Chat from the Chrome Web Store (when available) or load the unpacked extension in developer mode. Once installed, open the browser side panel and choose Sidebar AI Chat from the list of panels.

2. Add a provider

In the extension settings, pick Providers > Add provider. Choose the model provider you want to use (for example OpenAI or Ollama) and paste in the API key or base URL it requires.

3. Start your first chat

Create a new chat, select the provider you just added, and enter your prompt. The extension stores conversations locally, so you can return to them later.

You can now repeat the process for additional providers or configure advanced settings like custom system prompts.
