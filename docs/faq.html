General

Can I use multiple AI providers?

Yes. Add each provider in the settings and choose the one you need for every chat. You can switch models without losing conversation history.

Where are my chats stored?

Chats are saved locally in your browser. Clearing your cache or uninstalling the extension removes them, so export important conversations first.

Is there a cost to use Sidebar AI Chat?

The extension is free. You only pay for usage charged by the API providers whose keys you connect.

Does Sidebar AI Chat send data to external servers?

No. Requests go directly from your browser to the provider or proxy you configure. Ensure HTTPS is enabled on any proxy you expose to the public internet.

How do I find the extension ID mentioned in the proxy guide?
  1. Open chrome://extensions in your browser.
  2. Enable Developer mode in the top-right corner.
  3. Locate Sidebar AI Chat and copy the ID displayed beneath its title.
What does this extension do?

The extension adds a workspace where you can chat with AI assistants while keeping quick access to settings in a collapsible sidebar. It lets you create or select assistants, define reusable instructions, configure default models, review previous conversations, and attach files to messages.

Where does the data live?

All assistants, instructions, chat setups, conversation history, and uploaded files are stored locally in the browser using persistent storage. Your information stays on the device and is automatically restored the next time you open the extension.

Working with the Main Chat Area

How do I start a new chat?
  1. Select the + button below the message composer.
  2. Confirm that the workspace clears the current conversation.
  3. Send your next message to begin a fresh thread.
How can I review previous conversations?
  1. Click the history icon beside the new chat button.
  2. Browse the list of stored conversations.
  3. Select a thread to load it back into the workspace.
Can I upload files to a chat?

Yes. Drag files, images, or other supported assets onto the window. They are uploaded into local storage, linked to the relevant conversation, and displayed alongside the associated messages.

Sidebar & Settings Pages

How do I open the settings sidebar?
  1. Click the gear icon on the vertical toolbar.
  2. Wait for the sidebar to slide into view.
  3. Choose the section you want to manage from the sidebar navigation.
What sections are available in settings?
  • Assistants for creating and managing provider connections.
  • Instructions for reusable prompt templates.
  • Chat Configs for bundling assistants, instructions, and defaults.
How do I add a new assistant?
  1. Open Settings → Assistants.
  2. Select the + button at the bottom of the list.
  3. Fill in the assistant name, provider or model family, and required credentials.
  4. Save the form to make the assistant available for future chats.
How do I remove or edit an assistant?
  1. Open the assistant list from Settings → Assistants.
  2. Choose the delete control to remove an entry or open its detail view.
  3. Adjust the fields as needed—the changes save immediately.
What are instructions and how do I create them?

Instructions are reusable prompts that shape the behavior of your assistant. Navigate to Settings → Instructions to add, edit, or delete these templates. Once saved, instructions can be applied to any chat configuration.

What are chat configurations?

Chat configurations bundle an assistant, a set of instructions, and other preferences into a reusable preset. Manage them under Settings → Chat Configs to quickly switch the workspace between different workflows.

How do I switch between chat configurations?
  1. Hover over the chat icons on the vertical toolbar to find the configuration you need.
  2. Click a different icon to activate it.
  3. Confirm that the highlighted state updates, indicating the new defaults are applied.

Models & Providers

Which model providers are supported?

The extension supports multiple AI providers, including options such as OpenAI, Anthropic, xAI, and local runtimes. Each assistant can expose the models that are available for that provider, making it easy to compare responses across services.

How are models chosen in the chat UI?

When you open the message composer, the model selector presents the models from the currently active assistant or chat configuration. Switching configurations refreshes the list automatically so the selector stays in sync with your preferences.

Accessibility

Does the UI adapt when settings are open?

When the settings panel is active, the chat input is temporarily disabled. This prevents accidental sends while you adjust configurations and clearly indicates that the workspace is in edit mode.
